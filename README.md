# Web_Site_Vibecoding

## Pokémon SPA Game

SPA en JavaScript Vanilla (ES Modules) + Tailwind CSS CDN.

### Ejecutar localmente

```bash
python -m http.server 4173
```

Abrir `http://127.0.0.1:4173/index.html#/`.

### Rutas

- `#/` listado principal con búsqueda/filtro/paginación
- `#/pokemon/:id` detalle y captura/liberación
- `#/pokedex` pokédex personal con `localStorage`
- `#/battle` batalla RPG básica con selección de Pokémon capturados
